💡 Pipeline Technical Overview

All models were trained on the preprocessed Credit Card Approval Prediction dataset using 80/20 train/test splits. Categorical columns are handled dynamically via a serialized OneHotEncoder, and numerical columns are normalized with a StandardScaler. The models are packaged in complete, self-contained scikit-learn Pipeline objects to ensure zero feature leakages and seamless deployment.
